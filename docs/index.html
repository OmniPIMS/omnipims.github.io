Introduction

OmniPIMS is a schema-first engine that exposes dynamic REST APIs for any modeled entity.

Install

git clone https://github.com/OmniPIMS/omni-pims-backend.git
cd omni-pims-backend
npm install
cp .env.example .env
npm run dev

Schemas

Define attribute types, validations, and relationships. The engine stores schema definitions and exposes routes automatically.

Entities

After creating a schema, create entities using standard POST /entities routes. Validation follows the schema rules.

Docker

Example docker run:

docker build -t opims-backend .
docker run -p 3000:3000 opims-backend
