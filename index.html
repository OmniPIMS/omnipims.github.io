What is OmniPIMS?

OmniPIMS (OPIMS) is an open-source, schema-driven framework designed to store, define and manage highly dynamic entities. It allows organizations to model products, catalogs, insurance policies, properties, services and more — without modifying the backend.

Key Features

  • Dynamic schema definitions
  • Custom attributes and parameters
  • Entity management without code changes
  • Plug-and-play REST API
  • Docker-ready backend and frontend
